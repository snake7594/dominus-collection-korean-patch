# 캐슬바니아 도미누스 컬렉션 한글 개선패치 — Steam 최신판용 (v2)

**대상:** Castlevania Dominus Collection (Steam, 최신 빌드) — `windata\alldata.bin`
**형식:** xdelta3 (VCDIFF) · **패치 크기:** 약 29 KB

Steam 최신판에 이미 적용된 한글패치 위에, **번역 품질을 끌어올리는 개선 패치**입니다.

| 게임 | v2에서 개선되는 내용 |
|---|---|
| 오더 오브 에클레시아 (Order of Ecclesia) | 공식 한국어에 영문으로 남아 있던 **고유명사 915곳**(인명·아이템·마법·적·지명·BGM)을 한글 표기로 교체 |
| 빼앗긴 각인 (Portrait of Ruin) | 미번역으로 일본어가 노출되던 **자매모드 조작 설명** 등을 번역, 딸기 설명 등 오타/잔여 일본어 정리 |
| 창월의 십자가 (Dawn of Sorrow) | "어떤한→어떠한" 등 오타 수정 |

> ⚠️ 이 저장소에는 게임 데이터가 없습니다. 패치는 차분(diff)이며 **본인 소유의 게임 파일**에만 적용됩니다.

---

## ⚠️ 먼저 알아둘 것 — 이 패치는 "개선 레이어"입니다

이 패치는 **한글 베이스 패치가 적용된 상태**의 `alldata.bin` 위에 얹습니다.
즉 폰트와 대사 한글화는 베이스 패치가 담당하고, 이 v2는 그 위에서 **표기 개선·오타·미번역**만 바로잡습니다.

적용 순서:

```
① 최신 Steam 버전 게임               (pristine)
② + 한글 베이스 패치(Dra01/Dra02KoreanV4 적용)   ← 폰트 + 한글 대사
③ + 이 개선패치(DominusKorean_Steam_v2.xdelta)   ← dra03 이름 한글화 + 오타/미번역
```

②까지 끝난 `alldata.bin`(아래 해시)에 이 패치를 적용하세요.

## 준비물 / 해시

| 항목 | 값 |
|---|---|
| 파일 | `windata\alldata.bin` |
| 크기 | `1,279,606,784` bytes |
| 베이스(②, V4 적용) SHA1 | `ba2377dd7ce22896723b8de5c9000e067d1c8b95` |
| 적용 후(③) SHA1 | `9d04a391ce71b032934da3614990c6776135aad2` |

크기가 다르거나 베이스 해시가 안 맞으면 **게임 버전이 다르거나 베이스 한글패치가 적용되지 않은 것**입니다.

## 적용 방법

**방법 A — 동봉 스크립트 (권장, 해시 검증 포함)**

```bat
pip install pyxdelta
python apply_patch_steam.py "C:\Program Files (x86)\Steam\steamapps\common\Castlevania Dominus Collection\windata\alldata.bin"
```

원본을 `alldata.bin.bak` 으로 백업하고 개선본으로 교체합니다. 이미 적용된 파일이면 그냥 넘어갑니다.

**방법 B — Delta Patcher / xdelta3**

베이스 `alldata.bin` + `DominusKorean_Steam_v2.xdelta` → 출력 파일을 `alldata.bin` 으로 교체.

> 💡 Steam **"게임 파일 무결성 확인"** 을 돌리면 패치가 원상복구(재다운로드)됩니다. 적용 후에는 무결성 검사를 피하세요.

## 알려진 한계

- 베이스 한글패치(V4)가 폰트의 내부 코드 필드를 덮어써, 이 개선패치는 **새 글자 글리프를 폰트에 넣지 않습니다**(위험 회피).
  그래서 베이스 폰트에 없던 일부 글자(예: 몇·맨·칠·덴·떡)를 필요로 하는 자리는 원래 상태(일부 한자로 보임)를 그대로 둡니다.
  예: "몇번이고"가 "在번이고"로, "코나미맨"이 "코나미広"로 보이는 지점 — 이는 **베이스 패치 단계의 제약**이며 이 개선패치가 새로 만든 문제가 아닙니다.
- 에클레시아는 게임 자체의 완성형 한글 폰트를 쓰므로 위 제약이 없어 **전량 온전히 한글화**됩니다.
- HUD 약어(LV/EXP/ATK 등)와 영문 원제 BGM은 폰트 제약으로 영문 유지.
- **장비/글리프 메뉴의 일부 라벨**(Ｇｌｙｐｈ, Ｍａｉｎ／Ｓｕｂ／Ｂａｃｋ Ｇｌｙｐｈ, Ｇｌｙｐｈ Ｓｌｅｅｖｅ 등)은
  작은 UI 폰트(한글 미포함)로 그려져 한글로 두면 화면에 빈칸이 됩니다. 그래서 이 라벨들은 **영문으로 유지**합니다.
  (아이템 목록·설명·대사·지역명 등 큰 폰트로 그려지는 곳은 한글로 표시됩니다.)
