# 캐슬바니아 도미누스 컬렉션 한글 개선패치 — Steam 최신판용 (v2)

**대상:** Castlevania Dominus Collection (Steam, 최신 빌드) — `windata\alldata.bin`
**형식:** xdelta3 (VCDIFF)

이 패치는 **오더 오브 에클레시아(에클레시아의 후예)의 한국어 표기를 개선**하고,
**8px UI 폰트에 한글을 추가**하며, 창월/각인의 오타·미번역을 바로잡는 **개선 패치**입니다.

---

## ⚠️ 먼저 해야 할 것 — 베이스 한글패치(was564) 적용

이 패치는 **was564 님의 한글패치가 적용된 상태** 위에 얹는 "개선 레이어"입니다.
먼저 아래 베이스 패치를 적용하세요.

- **제작자:** **was564** (블로그: [놀이터 기록소](https://was564.tistory.com/9))
- **베이스 패치:** `Dra01KoreanV4-0.zip`(창월의 십자가) / `Dra02KoreanV4-0.zip`(폐허의 초상화)
  — 반드시 was564 공식 배포처(위 블로그의 구글 드라이브 링크)에서만 받으세요.
- **적용 순서(was564 안내):**
  1. 게임의 "로컬 파일 탐색" → `windata` 폴더에 zip 압축 해제
  2. `Dra01KoreanV4-0.exe` / `Dra02KoreanV4-0.exe` 실행
  3. **게임 언어를 일본어(JP)로 실행해야 한글로 플레이**됩니다
  4. 백신이 exe 실행을 막으면 예외 처리
- **NDS 원작 한글패치 감사(was564 님 표기):**
  창월의 십자가 — **aleksei, rURiKo, OgreMage** / 폐허의 초상화 — **KAISO, ex**

이 개선패치는 was564 님 패치의 상위 호환이 아니라 **그 위에 얹는 보완물**입니다.
번역·폰트 기반은 전적으로 was564 님과 원작 패치 팀의 작업입니다.

적용 순서 요약:

```
① 최신 Steam 게임 (pristine)
② + was564 한글패치(Dra01/Dra02KoreanV4)      ← 폰트 + 한글 대사 (필수 선행)
③ + 이 개선패치(DominusKorean_Steam_v2.xdelta) ← 에클레시아 한글화 + 8px 한글폰트 + 오타수정
```

## 준비물 / 해시

②까지 끝난 `windata\alldata.bin`(아래 베이스 해시)에 적용하세요.

| 항목 | 값 |
|---|---|
| 파일 | `windata\alldata.bin` |
| 크기 | `1,279,606,784` bytes |
| 베이스(②, was564 V4) SHA1 | `ba2377dd7ce22896723b8de5c9000e067d1c8b95` |
| 적용 후(③) SHA1 | `f8d4739dd1520459e53536e82e21e8f79eab2dcb` |

크기가 다르거나 베이스 해시가 안 맞으면 **게임 버전이 다르거나 was564 베이스 패치가 적용되지 않은 것**입니다.

## 적용 방법

**방법 A — 동봉 스크립트 (권장, 해시 검증 포함)**

```bat
pip install pyxdelta
python apply_patch_steam.py "C:\Program Files (x86)\Steam\steamapps\common\Castlevania Dominus Collection\windata\alldata.bin"
```

**방법 B — Delta Patcher / xdelta3**

베이스 `alldata.bin` + `DominusKorean_Steam_v2.xdelta` → 출력을 `alldata.bin`으로 교체.

> 💡 Steam **"게임 파일 무결성 확인"** 을 돌리면 원상복구되니 적용 후 피하세요.

## 이 개선패치(v2)가 바꾸는 것

- **8px UI에 한글 폰트 추가** ([Galmuri7](https://galmuri.quiple.dev/) 갈무리7 기반):
  기존엔 글리프 목록·메뉴 라벨이 작은 폰트에 한글이 없어 빈칸이었으나, 8px 한글 글리프를
  넣어 **글리프 이름·메뉴가 한글로 표시**됩니다. (콘포데레·유니온 세카레·메인/서브/백 글리프 등)
- **에클레시아 영문 표기 → 한글**: 인명·아이템·마법·적·지명·BGM·메인메뉴(장비/유물/지도/설정 등)
- **표기 통일**: 도미누스→**도미너스**, 바로우→**발로우**
- **창월/각인 오타·미번역 수정**: 어떤한→어떠한, 각인 자매모드 조작설명 번역 등

전체 변경 목록은 [CHANGES.md](CHANGES.md) 참고.

## 알려진 한계

- 베이스 폰트(V4)에 없던 일부 글자(몇·맨·칠 등)를 쓰는 창월/각인의 극소수 자리는
  원 상태(일부 한자로 보임)를 유지합니다 — 베이스 패치 단계의 제약.
- 화면 제목 그래픽("GLYPH" 등)과 스탯 약어(ATK/DEF 등)는 텍스트가 아니라 이미지/시스템 폰트라
  영문으로 유지됩니다.
